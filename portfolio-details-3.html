Información del proyecto

  • Categoria: API REST
  • Framework: .NET Core 3.1
  • Lenguaje de P.: C#

Este proyecto es una API REST y fue desarrollada en .NET Core 3.1.
y cuenta con los siguientes puntos:

  • Conexión a Base de Datos.
  • Arquitectura en capas y separación de proyectos.
  • Autenticación utilizando JWT.
  • Autorización con verificación de roles y/o perfiles.
  • Documentación automática con Swagger.
  • Inyección de dependencias para la instanciación de los repositories y services.
  • Pruebas unitarias.
